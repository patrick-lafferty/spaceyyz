SpaceYYZ is a single page application I wrote to learn and demonstrate front end development. It is a mockup of what a commercial satellite launch provider might use to manage their day to day operations. It currently uses:

  • AngularJS
  • Angular-UI
  • Angular-UI-Router
  • Bootstrap
  • Firebase

For tools I'm currently using:

  • npm
  • webpack
  • git
  • Less
  • Karma + Mocha + Chai
  • ESLint
